**создано 3 таблицы**

- **таблица (катридж)**
  - 1.1 id — автозаполнение через приложение
  - 1.2 название — в приложении при добавлении можно выбрать из существующих вариантов или нажать кнопку «Добавить новый катридж» (если такого названия нет в списке)
  - 1.3 статус — в приложении можно выбрать один из 4 статусов

- **таблица (склад)**
  - 2.1. катридж — создается автоматически на основе таблицы (катридж)
  - 2.2. количество — автоматически подсчитывается по названиям, совпадающим в таблице (катридж)

- **таблица (учёт)**
  - 3.1. картридж — заполняется вручную или выбирается из справочника картриджей
  - 3.2. количество — вводится вручную при оприходовании или списании
  - 3.3. дата начала — автоматически проставляется текущая дата при создании записи
  - 3.4. дата окончания — заполняется вручную при закрытии/завершении учёта
  - 3.5. категория — выбирается из предустановленного списка категорий
